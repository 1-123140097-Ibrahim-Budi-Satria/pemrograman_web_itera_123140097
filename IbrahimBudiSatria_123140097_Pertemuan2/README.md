# Aplikasi Personal Dashboard - Jadwal Kuliah

Aplikasi ini merupakan aplikasi web sederhana yang digunakan untuk membantu mahasiswa dalam mengelola jadwal kuliah. Aplikasi dibuat menggunakan HTML, CSS, dan JavaScript dengan penyimpanan data menggunakan localStorage browser.

## Aplikasi ini memiliki beberapa fitur, yaitu:

- Menambahkan jadwal kuliah baru
- Memperbarui data jadwal (edit)
- Menghapus jadwal yang sudah tidak diperlukan
- Melakukan filtering jadwal berdasarkan hari (Senin-Sabtu)
- Menampilkan statistik total mata kuliah dan kelas hari ini
- Menampilkan jam real-time

Semua data jadwal disimpan menggunakan localStorage. Penggunaan localStorage bertujuan agar data tetap tersimpan walaupun halaman web direfresh atau browser ditutup.
